Sublime Text and Stata Integration

StataEditor is a Sublime Text 3 plugin that integrates closely with Stata. It allows you to use Sublime Text's main features (syntax highlighting, autocompletion, snippets, build systems, goto symbols, etc.) interactively with Stata, so you can fully use the advantages Sublime's workflow in Stata.

Installation

Requirements

  • Stata 13 (should work up to Stata 11)
  • Sublime Text 3
  • Windows (it should be straightforward to add this for OSX using Andrew Heiss's work; not sure about Linux)

Installation

  • Install Package Control if not installed
  • Install the Pywin32 package (to install a package: within Sublime Text, press ctrl+shift+p, type "install", type the package name, and select it).
  • Add repository: ctrl+shift+p, type "add repo", and enter https://github.com/sergiocorreia/StataEditor
  • Install the StataEditor package.
  • Optional: Set the Stata Path: ctrl+shift+p, type "stata path" (or use the Tools - Stata menu). If this step is skipped, StataEditor will just try to find it, and ask if it fails.
  • Optional: Register the Stata Automation type library: ctrl+shift+p, "stata register", and accept the Windows Prompt. In case of an error, see the manual instructions. If this step is skipped, StataEditor will try to run it when required.

Using Stata

Example

Create a new file, and save it with a .do extension. It will recognize the Stata syntax in the lower left corner
